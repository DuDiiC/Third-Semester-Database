# Third Semester - Databases

Tasks from the laboratory: [Databases](https://usosweb.umk.pl/kontroler.php?_action=katalog2%2Fprzedmioty%2FpokazPrzedmiot&kod=1000-I1BADz&lang=pl). 

Tasks come from subject's moodle webside.

I used the [__Oracl/PLSQL__](https://www.techonthenet.com/oracle/index.php) syntax with __Oracle Database__.

---

### Issues discussed during the course:

- datatypes in databases
- creating tables with the `CREATE TABLE` statement
- adding new data into tables with the `INSERT INTO` statement
- searching the database with the `SELECT` statement, `WHERE`, `ORDER BY` `GROUP BY` clausule with `MIN`, `MAX`, `COUNT`, `SUM`
- basics [functions](https://www.techonthenet.com/oracle/functions/index.php) in PLSQL, e.g. `CONCAT`, `SUBSTR`, `ROUND`...
- primary and foreign keys (inline and with `CONSTRAINT` statement)
- `NOT NULL`, `DEFAULT`, `UNIQUE` and logical conditions while creating tables
- modyfying data in tables (`UPDATE`, `DELETE`) and modifying tables (`ALTER TABLE`, `DROP TABLE`)
- agregation functions (`GROUP BY` clausule with `MIN`, `MAX`, `COUNT`, `SUM`, `AVG`, `HAVING`)
- correlated and uncorrelated subqueries
- `JOIN` queries
- views, sequences
- grant and revoke privileges, roles
- transactions
